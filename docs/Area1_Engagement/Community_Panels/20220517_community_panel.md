
# TOPS Community Panel #1

**Where:** Public (virtual) meeting  
**When:** 17-19 May 2022, 12-3 EST (9-12 PST), each day.  
**How:** [Registration link](https://forms.gle/sgLgwM1nxGzqJbuA6)
